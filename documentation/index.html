User Guide

After installation of module, make sure the setup is upgraded and static-contents are deployed. Login to admin panel and flush you Magento cache storage.

How to Configure the Extension

At the backend go to “Rich Snippets”. You can go there from Admin Panel > FME Addons. Here you will find the following settings:

 

General Configuration:

  • Module Enable: Enable or disable module

 

Facebook Login Configuration:

  • Enable: Enable or disable
  • Application ID: Please enter facebook App id
  • Set Product SKU: Please enter facebook App secret key

 

Yahoo Login Configuration:

  • Enable: Enable or disable
  • Application ID: Please enter facebook App id
  • Consumer Key: Please enter yahoo sonsumer key
  • Consumer Secret: Please enter yahoo consumer secret

 

Google Login Configuration:

  • Enable: Enable or disable
  • Client Id: Please enter google client Id
  • Application name: Please enter google application name
  • Client secret: Please enter google client secret key

 

Twitter Login Configuration:

  • Enable: Enable or disable
  • Consumer Key: Please enter yahoo sonsumer key
  • Consumer Secret: Please enter yahoo consumer secret
